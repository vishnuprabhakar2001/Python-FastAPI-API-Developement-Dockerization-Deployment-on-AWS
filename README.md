# Python FastAPI API Development with Docker & AWS Deployment

This project demonstrates the complete development, containerization, and deployment of a Python **FastAPI** API using modern tools and practices.

## Overview

- Developed and designed the FastAPI application with the help of **AI tools (ChatGPT)** for guidance in architecture, coding, and optimization.  
- Created a **Dockerfile** to containerize the application, ensuring easy portability and consistent environment setup.  
- Built the Docker image locally and **pushed it to Docker Hub** for public/private access.  
- Deployed the application on an **AWS EC2 Ubuntu instance** by pulling the Docker image and running it as a container.  
- Verified the API functionality using **Postman**, ensuring endpoints work as expected.  

## Key Features

- FastAPI application with clean, modular code.  
- Containerized using Docker for easy deployment.  
- Hosted on cloud infrastructure (AWS EC2).  
- Tested using Postman for reliable API responses.
